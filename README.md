# Emotion Journal

帮助用户记录情绪、识别触发点、发现模式，建立温和的自我观察习惯

## What it does
- 引导用户按结构化格式记录情绪体验（事件、情绪、强度、触发点、思维模式）
- 帮助用户回顾多段情绪记录，识别重复出现的情绪模式
- 对模糊的内心不适进行梳理，帮助用户命名和理解自己的感受

## Example scenarios

**1. 记录工作中的负面情绪**
> 👤 帮我记一下今天的心情，今天在公司被批评了，心里很不舒服
> 🤖 好，我们先把这次情绪记下来。先说发生了什么…那一刻你最明显的情绪是什么？…按1到10分，这个感受大概有多强？…输出结构化反思：事件、情绪、强度、触发点、思维模式、下一步行动

**2. 情绪日记记录跟女朋友吵架**
> 👤 昨晚跟女朋友因为谁洗碗吵了一架，当时特别生气，现在想记录一下分析分析
> 🤖 引导按情绪日记格式记录：事件触发、情绪与强度（愤怒8/10，委屈4/10）、自动思维、替代思维、情绪强度复查（愤怒降到5/10），建议用手机备忘录或flomo记录

**3. 说不清自己怎么了**
> 👤 我也说不清自己怎么了，就是很不舒服，帮我整理一下
> 🤖 不用一次说得很完整，我帮你慢慢整理。最近发生了什么？最明显的身体感觉是什么？当时脑子里最强烈的想法是什么？输出帮你命名感受的反思总结，不强行贴标签
